## 1. 业务定义

**业务对象**：面向 TikTok 美区的 B2B 漫剧代加工生成平台。

**核心交付物**：50-60s 竖屏 9:16 漫剧单集成片。

**客户画像**：自有剧本的漫剧制作方，平台负责从剧本到成片的自动化生产。

**建设者**：1人（如意粑粑），懂代码和 AI，不懂漫剧专业流程。

**关键约束**：1周Demo目标、GPU走Vast.ai性价比路线、API质量优先但开源达标则首选。

## 3. 完整生产流程拆解

```
剧本输入(JSON/文本)
  → Step1: 剧本解析(LLM结构化)
  → Step2: 分镜生成(LLM + 规则引擎)
  → Step3: 角色资产(LoRA/IP-Adapter/参考图)
  → Step4: 批量生图(ComfyUI + Flux/SDXL)
  → Step5: 微动态制作(img2vid: Kling/Seedance/Runway)
  → Step6: 音频制作(ElevenLabs/Fish Audio + Suno + SFX)
  → Step7: 合成拼接(FFmpeg/DaVinci Free + 字幕 + 安全区)
  → Step8: 自动质检(CV规则 + 人工抽检)
  → Step9: 多版本导出(TikTok版 + C2PA元数据注入)
  → 交付
```

### 各环节自动化程度与人工介入点

| 环节 | AI自动化 | 必须人工 | 人工占比 |
|------|---------|---------|---------|
| 剧本解析 | 90%+ | ✅ 剧本创作 | 10% |
| 分镜设计 | 70-80% | ✅ 审核调整 | 20-30% |
| 角色资产 | 80% | ✅ 一致性审核 | 20% |
| 图像生成 | 95% | ✅ 选图/微调 | 5% |
| 微动态/视频 | 85-90% | ✅ 质量筛选(3-5次/镜头) | 10-15% |
| 音频生成 | 90% | ⚠️ 配音情感审核 | 10% |
| 合成剪辑 | 60-70% | ✅ 创意剪辑、节奏 | 30-40% |
| 质检 | 70% | ✅ 最终审核 | 30% |
| 导出 | 99% | ❌ | 1% |

**整体人工占比**：约30-40%，目标<10%在当前技术下不可行 ✅ 已验证

## 10. 不可行项与高风险项

### ❌ 已知不可行项

| 不可行项 | 说明 |
|---------|------|
| 一次性生成>2分钟连贯视频 | 当前最长单次生成10秒（Kling），需多clip拼接 |
| 完美跨镜头角色一致性 | 所有模型跨镜头仍有漂移，LoRA是最接近的方案(8.5/10) |
| 全自动叙事节奏把控 | AI无法替代人类对叙事节奏的判断 |
| 10分钟+连贯短剧无人工干预 | 需大量拼接和一致性维护 |
| <10%人工介入 | 实际约30-40%，创意和质量把关必须人工 |
| S级≤$5/集 | 仅Kling视频一项就超$8/集 |
| XTTS-v2商用 | 项目已停止+非商用许可 |

### 🔴 高风险项

| 风险项 | 严重程度 | 应对 |
|--------|---------|------|
| InsightFace非商用许可 | 🔴 高 | InstantID商业使用必须购买许可，或改用LoRA |
| API提供商锁定 | 🔴 高 | Sora已关停证明风险；多模型策略+统一API平台 |
| TikTok未标注AI内容 | 🔴 高 | 所有内容主动标注C2PA+invisible watermark |
| EU AI Act合规(8月2日) | 🔴 高 | 机器可读标记强制性要求 |
| Seedance/IP审核 | 🟡 中 | 使用原创角色设计，避免与现有IP相似 |
| Suno商用许可覆盖TikTok（来源：batch-04.md §9.4） | 🟡 中 | 优先使用Veo原生音频或确认Suno许可条款 |

| **单集成本** | **$1-4（A级）** | 可控 |

### 11.2 后续迭代路线

1. **Week 2-3**：升级S级流水线（LoRA微调 + Kling标准），成本约$9-13/集
2. **Week 4-6**：升级SSS级流水线（**默认路径B**，保留路径A备选，新增SSSS容错 tier）：
   - **路径B（默认 ✅ UC-2）**：LoRA + Seedance 2.0 Fast img2vid，**目标$8-10/集**
     - Seedance 2.0 Fast $0.022/sec，当前最低价商用方案
     - ⚠️ 风险：ByteDance审核严格，需原创角色设计
   - **路径A（备选）**：LoRA + Kling 3.0 img2vid，**目标$14-18/集**
   - **SSSS tier（降级容错 ✅ UC-3）**：IP-Adapter + Seedance 2.0基础版，**目标$5-7/集**
     - 当路径B或路径A均失败时触发，保留LoRA资产复用
3. **降级容错链**（已确认 UC-3 ✅）：
   ```
   SSS路径B(Seedance $8-10) → P0验证失败 → SSS路径A(Kling $14-18)
       → 再失败 → SSSS(IP-Adapter + Seedance $5-7)
           → 还失败 → A级兜底(IP-Adapter + Pika 2.0, $1-4)
   
   故障恢复：每个层级保留上一版本资产（LoRA/参考图），降级无需重新训练
             视频生成失败3次自动切换备选模型（Kling↔Seedance）
   ```
4. **探索降本**：Fish Audio S2 Pro替代ElevenLabs（$0.55→$0.05/集），Seedance 2.0替代部分Kling
5. **长期**：开源视频模型自部署降本70-85%
