### Phase 1 — A 级 Demo 流水线 + 生产调度面板

| 要素 | 内容 |
|------|------|
| **目标** | 后端实现 A 级完整流水线（角色资产→视频→音频→合成），前端实现生产调度面板（一键生产+实时进度+素材浏览） |
| **输入** | ①Phase 0 环境和 Web 骨架 ②剧本素材 |
| **输出** | ①A 级流水线后端代码（IP-Adapter + Kling img2vid + ElevenLabs TTS + FFmpeg 合成）②Web UI 生产调度面板（一键生产按钮+实时进度 WebSocket 推送）③素材浏览器（角色资产+镜头素材按集浏览）④1 集 A 级 Demo 成片 ⑤单集成本报告（目标 $1-4） |
| **验收标准** | ①通过 Web UI 一键启动 A 级生产，无需命令行操作 ②生产进度实时显示（每集每镜头状态）③素材浏览器可查看所有生成资产 ④Demo 成片质量 A 级（一致性 ≥0.80）⑤单集成本 ≤ $4 ⑥C2PA 元数据注入验证通过 |
| **依赖关系** | Phase 0 ✅；**阻塞项 #1 和 #3 必须在本 Phase 启动前验证通过** |

**阻塞项处理**：
- **#1 IP-Adapter 一致性**：Phase 1 编码前验证，失败则降级为 Kling 角色锁定
- **#3 Kling 审核误判率**：与 #1 并行验证，失败则切换 Seedance 2.0 Fast

**编码任务**：
1. 后端：实现角色资产生成模块（IP-Adapter Plus / ComfyUI pipeline）
2. 后端：实现 Kling 3.0 img2vid 视频生成模块（prompt 模板、重试逻辑）
3. 后端：实现 ElevenLabs Creator TTS 音频生成模块（角色 voice mapping）
4. 后端：实现 FFmpeg 合成模块（视频拼接、音频叠加、字幕烧录、安全区处理）
5. 后端：实现 C2PA 元数据注入模块
6. 后端：实现生产调度 API（启动/暂停/取消/重跑、WebSocket 进度推送）
7. 前端：实现生产调度面板（项目工作台、一键生产、实时进度、任务队列）
8. 前端：实现素材浏览器（角色资产卡片、镜头素材按集浏览）
9. 执行 Demo 生产，产出 1 集成片，记录成本和耗时

## 风险评估

| 风险 | 级别 | 影响 | 应对措施 |
|------|------|------|---------|
| #1 IP-Adapter 一致性不达标 | 🟡 中 | A 级流水线需降级 | 降级为 Kling 角色锁定 |
| #2 LoRA→Seedance 衔接失败 | 🟡 中 | SSS 路径 B 不可用 | 触发降级链，路径 A 为默认 |
| #3 Kling 审核误判率高 | 🟡 中 | Demo 视频生成受阻 | 切换 Seedance 2.0 Fast |
| #4 Suno 许可不覆盖 TikTok | 🟢 低 | BGM 备选方案受限 | 严格用 Veo 3.1 原生音频 |
| Web UI 前端工作量大 | 🟡 中 | Phase 5 可能延期 | Gradio 过渡 + 分优先级迭代 |
| API 服务不稳定 | 🟡 中 | 生成失败率上升 | 重试机制 + 降级链自动切换 |
| GPU 资源不足 | 🟡 中 | LoRA 训练延迟 | Vast.ai 弹性扩缩 |
