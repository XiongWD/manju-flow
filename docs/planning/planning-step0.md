## 6. 降级容错链确认

```
SSS路径B(Seedance $8-10) → P0验证失败 → SSS路径A(Kling $14-18)
    → 再失败 → SSSS(IP-Adapter + Seedance $5-7)
        → 还失败 → A级兜底(IP-Adapter + Pika 2.0, $1-4)

故障恢复：每个层级保留上一版本资产（LoRA/参考图），降级无需重新训练
          视频生成失败3次自动切换备选模型（Kling↔Seedance）
```

**结论**：✅ 接受（UC-3已确认）
